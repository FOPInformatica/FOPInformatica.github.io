FOPI main page


Check whether you have Ruby 2.1.0 or higher installed:

    $ ruby --version
    > ruby 2.X.X


Install Bundler:

    $ gem install bundler
    # Installs the Bundler gem
    
    
Install the gem dependencies:

    $ bundle install
    
    
Serve it:

    $ bundle exec jekyll serve
    
And go to http://127.0.0.1:4000/ to see the site
